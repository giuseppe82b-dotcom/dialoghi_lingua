Guida per il Docente 🍎

Per creare una lezione, salva un file con estensione .json seguendo questa struttura:

  • personaggio: Il nome di chi parla.
  • italiano: La frase mostrata nel fumetto.
  • inglese: La traduzione corretta (parola per parola).
  • extra: Parole sbagliate per confondere lo studente.

Esempio di codice:

[
 {
  "personaggio": "Anna",
  "italiano": "Ciao, come stai?",
  "inglese": "Hi how are you",
  "extra": ["apple", "red"]
 },
 {
  "personaggio": "Marco",
  "italiano": "Bene grazie!",
  "inglese": "Fine thank you",
  "extra": ["blue", "cat"]
 }
]
